# Security Policy

Sentinel Guard is a research/work-sample prototype. The built-in demo tools are
simulated and should not be treated as a production security boundary.

Please do not include secrets, real credentials, private `.env` files, or
production traces in issues or pull requests.

For security-sensitive reports, contact the repository owner privately instead
of posting exploit details publicly. If you fork or adapt this project for real
agent tooling, review the policy, hook configuration, audit log location, and
all tool adapters before relying on it.
